正则表达式基础

正则表达式基础

+五大类

  • 预定义类
    • /./ 除了\n、\r、“”，都可以匹配
    • \d：匹配数字，\D：匹配非数字
    • \s：匹配空格， \S：匹配非空格
    • \w：匹配字母（包括_），\W：匹配非字母
  • 简单类
    • /字符串/：要有一个完整的字符串才能被匹配，可以多加字符，但不能减少，顺序也不能颠倒，有他就行
    • /[ 字符串 ]/：有字符串的任意一个字符都可以
  • 负向类
    • /[ ^abc ]/：只要有abc这个字符串（或者其中的一个字符）就返回false
  • 范围类
    • /[ 0-9 ]/
  • 组合类
    • / [ 0-5a-d ] /

+边界

  • ^$：同时用表示必须精确匹配

+量词

  • *：重复0次或者多次
  • +：重复1次或者多次
  • ？：重复0次或者1次
  • ｛｝：重复几次
    • ｛n｝：重复 n 次
    • ｛n，｝：最少重复 n 次
    • ｛n，m｝：最少n次，最多m次
  • |：或者，前面的或者后面的
  • （）：提高运算优先级

+匹配模式

  • g：全局模式
  • i：不区分大小写模式
  • m：匹配换行

+匹配中文

  • 匹配中文：/^[\u4e00-\u9fa5]{2,4}$/

+API

  • exec()
    • reg.exec(str)
    • 字符串中指定的值（）带括号的就是指定的值。返回找到的值，并确定其位置。
    • 找到了匹配的文本，则返回一个结果数组,否则返回null
  • match()
    • str.match(reg)
    • 如果正则是全局匹配，返回字符串中所有指定的值，成为数组
  • test()
    • reg.test(str)
    • 检索字符串中指定的值。返回 true 或 false。
